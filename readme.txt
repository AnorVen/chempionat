Дан список футбольных матчей, разбитый по турнирам
 (массив данных можно взять из файла data.txt). Надо вывести весь этот список, согласно
 приложенной картинке (layout.png). По верстке никаких особых требований нет - шрифты, цвета, отступы можно использовать произвольные.

В строке матча со статусом "не начался" выводится чекбокс, при клике на который матч добавляется в
 избранное или удаляется из избранного. При перезагрузке страницы состояние чекбоксов должно сохраняться (localStorage).

Кроме того над списком расположены две вкладки: "Сегодня" и "Избранное".
В скобках у "Сегодня" выводится общее число матчей, у "Избранного" - количество избранных матчей.
При клике на вкладку "Избранное" происходит фильтрация - отображаются только избранные матчи.
При перезагрузке страницы всегда активна вкладка "Сегодня".

Задачу следует реализовать на нативном JavaScript c применением любого шаблонизатора.
